arrancar servidor:  manage.py runserver
Crear una app:   manage.py startapp nombre
comprobar app:   manage.py check nombre
crear migraciones:   manage.py makemigrations
crear sql:    manage.py sqlmigrate nombre-app numero-migracion
subir migraciones a la base de datos:   manage.py migrate
abrir shell python:   manage.py shell   salir:   exit()
